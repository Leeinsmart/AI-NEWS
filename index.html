核心概念：2026 最大的 Prompt Engineering 創新是 AI Agent 鏈——將單一行銷任務拆成多個 Agent 串聯：User Prompt → 研究 Agent → 摘要 Agent → 寫手 Agent → 編輯 Agent，每個 Agent 有獨立角色和品質閘門。

操作步驟：① 定義行銷目標和交付物 ② 將任務拆解為 3–5 個 Agent 角色（Research → Synthesize → Write → Edit → QA）③ 每個 Agent 給予獨立 System Prompt、輸入格式和輸出規範 ④ 設定品質閘門：前一個 Agent 輸出需通過條件檢查才進入下一個
